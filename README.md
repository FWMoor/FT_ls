# FT_ls - C - 2019  
FT_ls is a recreation of the unix function, ls.  

## Installation:

To get started, clone the FT_ls repo and go into the folder:

```
git clone https://github.com/FWMoor/FT_ls
cd FT_ls/FT_ls
```
## Make:    

```
make
```

### Usage:

```
./ft_ls [-AGRaflnprt1] [file ...]  
```
